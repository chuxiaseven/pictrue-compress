# 图片压缩工具 - 技术架构图

## 1. 整体架构流程图

```mermaid
graph TB
    subgraph "用户层"
        U[用户界面]
    end

    subgraph "前端应用层"
        FE[Vue 3 + Vite]

    end

    subgraph "后端服务层"
        BE[Node.js + Express]

        FS[文件系统<br/>临时文件/历史记录]

    FE -->|生产模式| Electron
    Electron -->|HTTP 请求| BE
    ViteDev -->|HTTP 请求| BE
    BE -->|图片处理| Sharp
    BE -->|ZIP 打包| Archiver
    BE -->|读写| FS
    Sharp -->|读写| FS
    Archiver -->|读写| FS
```

## 2. 前后端交互流程图

```mermaid
sequenceDiagram
    participant User as 用户
    participant Frontend as 前端(Vue)
    participant Backend as 后端(Node.js)
    participant Sharp as 图片处理(Sharp)

    User->>Frontend: 1. 选择/拖拽图片
    Frontend->>Frontend: 2. 校验文件格式和大小
    Frontend->>Frontend: 3. 构建 FormData
    Frontend->>Backend: 4. POST /api/upload
    Backend->>Backend: 5. 校验文件
    Backend->>Backend: 6. 保存临时文件
    Backend-->>Frontend: 7. 返回文件信息
    Frontend->>User: 8. 显示上传成功

    User->>Frontend: 9. 配置压缩参数
    Frontend->>Backend: 10. POST /api/compress
    Backend->>Backend: 11. 读取临时文件
    Backend->>Sharp: 12. 解码图片
    Sharp->>Sharp: 13. 尺寸调整
    Sharp->>Sharp: 14. 质量压缩
    Sharp->>Sharp: 15. 格式转换
    Sharp->>Sharp: 16. 元数据去除
    Sharp->>Backend: 17. 返回处理后图片
    Backend->>Backend: 18. 保存压缩文件
    Backend-->>Frontend: 19. 返回压缩结果
    Frontend->>User: 20. 显示压缩结果

    User->>Frontend: 21. 查看对比
    Frontend->>Frontend: 22. 并排/滑块对比
    Frontend->>User: 23. 显示对比效果

    User->>Frontend: 24. 下载压缩图
    Frontend->>Backend: 25. GET /api/download
    Backend->>Backend: 26. 读取压缩文件
    Backend-->>Frontend: 27. 流式返回文件
    Frontend->>User: 28. 下载完成
```
